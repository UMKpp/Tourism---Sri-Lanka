# Tourism - Sri Lanka

**Live Demo:** [https://tourism-sri-lanka.vercel.app/](https://tourism-sri-lanka.vercel.app/)

## About the Project
We two built this project together to help solve tourism problems in Sri Lanka.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Vanilla), and JavaScript.
- **Deployment:** Vercel.

## System Architecture
This project is built as a static web application. The architecture consists of:
- **Client Side (Browser):** Renders the static HTML, applies CSS styling, and executes JavaScript for interactivity.
- **Hosting (Vercel):** Acts as a web server and Content Delivery Network (CDN), directly serving the static assets to the user's browser without a backend server or database.
